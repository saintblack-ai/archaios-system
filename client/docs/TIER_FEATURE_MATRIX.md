# Tier Feature Matrix

| Capability | Free | Pro | Elite |
|---|---|---|---|
| Access to landing/pricing | Yes | Yes | Yes |
| Dashboard preview | Limited | Full | Full |
| Daily intelligence brief | Delayed/limited | Full | Full + priority lane |
| Premium agent outputs | No | Yes | Yes |
| Core dashboard actions | Locked | Unlocked | Unlocked |
| Analytics visibility | Basic preview | Standard full | Full depth/priority |
| Priority signal stream | No | No | Yes |
| Upgrade prompts | To Pro/Elite | To Elite | Minimal |

## Restriction Logic
- Free:
  - Read-only preview posture for premium areas.
- Pro:
  - Full operational posture, excluding Elite-only priority stream/depth.
- Elite:
  - Complete access to all currently defined features.
