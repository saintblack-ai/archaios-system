# ARCHAIOS Agent Workspace

This folder is the control boundary for ARCHAIOS infrastructure work.

The ARCHAIOS agent owns system architecture, routing, dashboards, export intake, deployment readiness, billing infrastructure, repo maps, and monitoring reports. It does not create promotional content, social captions, campaigns, or book-marketing assets.

Operational source files remain in their existing locations so live imports, routes, and workflows do not break. This folder tags and indexes the infrastructure mission.

## Allowed Work

- Maintain infrastructure docs and system maps.
- Maintain export intake plans and scripts.
- Monitor GitHub, Cloudflare, Vercel, Supabase, Stripe, routes, and dashboards.
- Improve routing, auth skeletons, pricing infrastructure, dashboards, and operator/admin shell.
- Prepare deployment checklists and environment variable maps.

## Not Allowed

- Do not generate marketing content.
- Do not schedule or post social content.
- Do not modify Book Growth campaign copy, content packs, or audience hooks.
- Do not deploy, push, edit DNS, or change billing without explicit approval.
