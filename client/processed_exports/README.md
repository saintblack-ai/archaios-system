# ARCHAIOS Processed Exports

This layer contains cleaned/indexed derivatives from raw exports.

Subfolders:
- `indexes/` machine + markdown inventory outputs
- `knowledge_snapshots/` categorized knowledge extracts
- `project_snapshots/` project-oriented summaries
- `revenue_snapshots/` monetization prep summaries
- `review/` duplicate/overlap copies queued for review

Rules:
- Keep source references to raw exports.
- Never treat processed copies as original legal records.
